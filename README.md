# Spam Classifier Web App

## 📌 Overview
A Flask web app that classifies emails as **Spam** or **Not Spam**.  
It includes:
- A form to enter subject and message
- A classifier that predicts spam/not spam
- A history page with charts and CSV download

## 🚀 Features
- Classify emails instantly
- View classification history
- Download history as CSV
- Visualize spam vs not spam with charts
